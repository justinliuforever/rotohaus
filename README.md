# ROTOHAUS Residency

烟火邛州 - 国际非遗艺术驻地网站

## 启动

```bash
npm install    # 首次运行
npm run dev    # 开发模式 → http://localhost:5173
npm run build  # 生产构建
npm run preview # 预览生产版本
```

## 技术栈

- React + Vite
- Tailwind CSS v4

## 结构

```
src/
├── components/
│   └── ResidencyCover.jsx  # 封面 section
├── App.jsx
└── index.css
public/
└── images/                 # 图片资源
```

## 字体
